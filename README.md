# ArbolJS

Pagina romantica con un arbol de corazones animados y la frase *"Eres mi persona Favorita"*.

## Ver en local

Abre `index.html` en el navegador o ejecuta:

```powershell
cd C:\Users\admin\Desktop\ArbolJS
py -m http.server 8080
```

Luego visita `http://localhost:8080`.

## Publicar en GitHub Pages

1. Sube este repositorio a GitHub.
2. Ve a **Settings → Pages**.
3. En **Build and deployment**, elige **Deploy from a branch**.
4. Branch: **main**, carpeta: **/ (root)**.
5. Guarda y espera 1–2 minutos.

Tu sitio quedara en:

`https://TU-USUARIO.github.io/ArbolJS/`
